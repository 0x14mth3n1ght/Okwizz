# README

## Structure théorique du porojet

```
pima2022-groupe9
├── README.md
└── src
    ├── models # contien les models
    │   └── model.php
    ├── public # contien les controlers (charges des views en fonctions des besoins)
    │   ├── index.php
    │   ├── pseudo.css
    │   ├── pseudo.html
    │   └── stylesheet.css
    └── view # contien les views qui font appel au controleurs
        ├── footer.php
        ├── home.php
        ├── navbar.php
        └── question.php
```
